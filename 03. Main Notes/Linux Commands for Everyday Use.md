# Linux Commands for Everyday Use

## Idea

Everyday Linux commands are the ones you reach for without thinking — the commands where muscle memory pays off. They form the foundation of shell fluency: navigating the filesystem, inspecting files, and orienting yourself in the system.

## Why it matters

These commands are the highest-value ones to master because you use them dozens of times per session. Investing in knowing their flags and nuances pays compounding returns — every second saved on navigation or inspection is a second you can spend on the actual problem.

## Core Commands

- [[Linux cd command]] — Change the current directory; the primary navigation primitive
- [[Linux cat command]] — Concatenate and display file contents
- [[The pwd command]] — Print the working directory; reorient when you're lost
- [[Linux ls command]] — List directory contents with sorting and filtering flags
