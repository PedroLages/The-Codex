Read `90. AI/skills/session-handoff.md` and follow its workflow.

Create a session handoff in `90. AI/history/session-YYYY-MM-DD.md` (use today's date).

## What to Include

- **Session goal** — what Pedro wanted to accomplish
- **What was done** — tasks completed, progress made
- **Files created** — list new files with paths
- **Files updated** — list modified files with paths
- **Files backed up** — any backups made before edits
- **Decisions made** — key choices and why
- **Pending Pedro approval** — drafts, proposals, changes waiting for review
- **Open questions** — things not yet resolved
- **Next steps** — what to do when the work continues
- **Important context** — anything the next AI session needs to know
- **Related files** — key files for continuing the work

## Rules

- This is not memory. It's a session continuity note.
- If something should become memory, suggest running `/memory-proposal` separately.
- Put the handoff in `90. AI/history/`, not in `generated/`.
- Keep it concise — the next AI session needs the gist, not a transcript.
