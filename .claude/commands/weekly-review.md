Run a weekly review for The Codex. Create the report in `90. AI/generated/weekly-review-YYYY-MM-DD.md` (use today's date).

## Review These Areas

1. **`01. Inbox/`** — What's waiting to be processed? What's stuck?
2. **`90. AI/generated/`** — What AI drafts are waiting for Pedro's review?
3. **Recent notes in `03. Main Notes/`** — What was created or improved this week?
4. **`02. Source Materials/`** — What was consumed? What still needs processing?
5. **`05. MOCs/`** — Which MOCs need updating based on new notes?
6. **Open AI drafts** — Any drafts outside `generated/` that need attention?
7. **Memory** — Any stable context that changed and warrants a memory proposal?

## Memory Guardrail

Do not edit `90. AI/memory.md`. If memory should be updated, note it in the report and suggest running `/memory-proposal` separately.

The memory workflow is: Weekly Review → Memory Proposal → Pedro Approval → memory.md Update → memory-changelog.md entry.
