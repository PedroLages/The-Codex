# Session Handoff Skill

## Purpose

Create a clear handoff note at the end of a long AI work session.

This helps Pedro continue later without losing context.

This is not memory.

This is not a daily log.

This is a project/session continuity note.

## When To Use

Use when Pedro says:

- create handoff
- summarize this session
- save where we are
- create next steps
- close this session
- what did we change?
- prepare handoff for next AI session

Also suggest using it after:

- long DeepSeek sessions
- vault structure changes
- AI system changes
- big note-processing sessions
- weekly reviews
- project planning sessions

## Output Location

Create handoff notes in:

```text
90. AI/history/
```

Filename format:

```text
YYYY-MM-DD HHMM - Session Handoff - Topic.md
```

## Rules

* Keep it factual.
* Do not invent changes.
* Do not put temporary details into memory.
* If something should become memory, propose it through the Memory Curator skill.
* Link to changed files when possible.
