# Daily Brief Skill

## Purpose

Help Pedro start the day with clarity.

Use this skill to create a simple daily focus note.

## When to Use

Use when Pedro asks:

- daily brief
- morning brief
- what should I focus on today?
- plan my day
- start my day
- today's focus

## Output Location

Default output:

`90. AI/generated/`

## Output Format

```markdown
# Daily Brief - YYYY-MM-DD

## Today's Main Focus

1.
2.
3.

## Open Loops

-

## Learning Focus

-

## Work / Life Notes

-

## Notes To Process

-

## One Thing To Avoid

-

## End-of-Day Question

What did I learn today?
```

## Rules

* Keep it short.
* Do not overload the day.
* Choose 1 to 3 main priorities.
* Include learning progress when relevant.
* Include notes to process when relevant.
* Do not invent calendar, email, or task information unless it is available.
