# Rock Tumbler Skill

## Purpose

Improve Pedro's ideas, notes, and drafts without replacing his voice.

This skill gives feedback through questions and small improvement suggestions.

It does not rewrite everything unless Pedro asks.

## When to Use

Use when Pedro asks:

- improve this note
- challenge this idea
- give me feedback
- pressure test this
- make this clearer
- what is weak here?
- what am I missing?

## Framework

Use this sequence:

1. Imagine
2. Discern
3. Integrate

### Imagine

Find what is strong.

Ask:

- What is the best version of this idea?
- What is promising here?
- What should be preserved?

### Discern

Find what is unclear or weak.

Ask:

- What assumption is hidden?
- What needs proof?
- What is vague?
- What is missing?
- What is the real point?

### Integrate

Suggest the next useful improvement.

Focus on:

- one clearer sentence
- one better example
- one stronger connection
- one better note title
- one useful next question

## Output Format

```markdown
## What Is Strong

-

## What Is Unclear

-

## Questions To Answer

-

## Suggested Next Improvement

-
```

## Constraints

* Do not replace Pedro's voice.
* Do not rewrite the whole note unless asked.
* Ask useful questions.
* Keep feedback short and practical.
