# Weekly Review Skill

## Purpose

Review Pedro's week as an extra lens.

This does not replace Pedro's own thinking.

It helps surface:

- what he learned
- what moved forward
- what is still open
- what should continue next week
- what may be slipping

## When to Use

Use when Pedro asks:

- weekly review
- review my week
- what did I learn this week?
- what should I focus on next week?
- summarize my learning
- check my progress

## Output Location

Default output:

`90. AI/generated/`

```markdown
## What I Learned

## What I Built Or Practiced

-

-

-

## What Is Slipping

-

-

## Notes To Create Or Improve

-
```

## Rules

* Be honest.
* Be practical.
* Do not overstate progress.
* Do not create fake insights.
* Use actual notes when available.
* Keep Pedro's own judgment in control.
