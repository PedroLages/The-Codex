# Linux MOC

## Overview

Linux is the open-source operating system kernel that powers most servers, cloud infrastructure, and embedded devices. Understanding Linux means understanding how to interact with the system through the shell, navigate the filesystem, manage processes, and compose commands to solve real problems.

## Core Notes

- [[Linux Shell]] — The command-line interpreter that translates text instructions into OS operations. The primary interface for remote management and headless servers.
- [[Linux Commands for Everyday Use]] — The most frequently used commands: `ls`, `cd`, `pwd`, `cat`.
