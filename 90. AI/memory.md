This is not a dump file.

Only add information that is useful again in the future.

# Memory Rules

## What Belongs Here

Add only:

- stable facts
- stable preferences
- vault decisions
- naming decisions
- AI workflow decisions
- active project context
- long-term learning focus
- repeated patterns that are useful for future work

## What Does Not Belong Here

Do not add:

- temporary thoughts
- one-time tasks
- raw notes
- source material
- long summaries
- random AI outputs
- unapproved assumptions
- things that belong in Main Notes
- things that belong in Source Materials
- daily diary entries

## Update Rule

The official memory workflow is:

```text
Weekly Review → Memory Proposal → Pedro Approval → memory.md Update → memory-changelog.md entry
```

AI must not update this file automatically.

AI must create a memory update proposal first.

Memory update proposals go in:

```text
90. AI/generated/
```

After Pedro approves, AI may update this file safely.

Every approved update must also be recorded in:

```text
90. AI/memory-changelog.md
```
