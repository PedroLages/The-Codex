# 03 - Create Main Note Draft

Use this to create a draft main note from an idea or source.

```text
Create a draft Main Note from this idea or source:

`PASTE_IDEA_OR_FILE_PATH_HERE`

Use my Main Note style.

The note must:

- focus on one idea
- have a clear statement-style title
- be written in simple language
- explain the idea in my own words
- explain why it matters
- include one concrete example
- suggest tags as `[[Tag]]`
- suggest related notes
- suggest a MOC if useful
- link back to source material if available

Put the draft in `90. AI/generated/`.

Do not move it to `03. Main Notes/` yet.

After the draft, include:

- suggested final file name
- suggested folder
- quality score from 0-10
- what Pedro should review before approving
```
