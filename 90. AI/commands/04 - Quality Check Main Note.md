# 04 - Quality Check Main Note

Use this before promoting a note into Main Notes or improving an existing Main Note.

```text
Use `90. AI/skills/note-quality-checker.md`.

Review this note:

`PASTE_FILE_PATH_HERE`

Do not edit the note yet.

Check:

1. Is it one idea?
2. Is the title clear?
3. Is it written in Pedro's own words?
4. Is it understandable without the source?
5. Does it explain why the idea matters?
6. Does it include a useful example?
7. Does it link to source material?
8. Does it use useful tag links?
9. Does it connect to a MOC if needed?
10. Should it be split, improved, or promoted?

Create a quality report in `90. AI/generated/`.

Return:

- status: Ready / Needs Work / Not a Main Note Yet
- score from 0-10
- what is strong
- what needs work
- suggested edits
- better title options
- suggested tags
- suggested MOC links
- next action for Pedro
```
