# 01 - Process Inbox

Use this to review and process Inbox notes safely.

```text
Use `90. AI/skills/inbox-processor.md`.

Review `01. Inbox/`.

Do not move, delete, rename, or edit any notes yet.

Create a processing proposal in `90. AI/generated/`.

For each inbox note, suggest one action:

1. Keep in Inbox
2. Turn into Main Note
3. Move to Source Materials
4. Link to existing note
5. Needs Pedro review

For each note, include:

- current file name
- what the note seems to be about
- recommended action
- reason
- suggested title if it becomes a Main Note
- suggested tags using `[[Tag]]`
- suggested MOC if useful
- next action for Pedro

At the end, give me:

- notes ready to process
- notes needing my decision
- possible duplicates
- suggested next steps

Do not make changes. Proposal only.
```
