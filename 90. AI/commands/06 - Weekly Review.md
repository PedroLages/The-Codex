# 06 - Weekly Review

Use this once per week.

```text
Use `90. AI/skills/weekly-review.md`.

Run a weekly review for The Codex.

Review:

- `01. Inbox/`
- `90. AI/generated/`
- recent notes in `03. Main Notes/`
- source notes in `02. Source Materials/`
- current MOCs in `05. MOCs/`
- open AI drafts
- possible memory changes

Create the weekly review in `90. AI/generated/`.

Include:

1. What I learned
2. What I created
3. Best notes created or improved
4. Inbox notes that need processing
5. AI drafts that need approval
6. MOCs that need updating
7. Open loops
8. What is slipping
9. Focus for next week
10. Suggested memory update proposal if stable context changed

Do not edit `90. AI/memory.md`.

If memory should be updated, create a proposal only.

Use this workflow:

Weekly Review → Memory Proposal → Pedro Approval → memory.md Update → memory-changelog.md entry
```
