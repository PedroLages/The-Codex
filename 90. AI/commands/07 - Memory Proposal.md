# 07 - Memory Proposal

Use this when something important should maybe be added to memory.

```text
Use `90. AI/skills/memory-curator.md`.

Review this context:

`PASTE_CONTEXT_OR_FILE_PATH_HERE`

Create a memory update proposal.

Do not edit `90. AI/memory.md`.

Put the proposal in `90. AI/generated/`.

Use this workflow:

Memory Proposal → Pedro Approval → memory.md Update → memory-changelog.md entry

Classify possible memory items as:

- Add
- Update
- Outdate
- Ignore
- Needs Pedro Approval

Only propose stable context that will be useful in the future.

Do not add:

- temporary thoughts
- raw notes
- source material
- AI guesses
- one-time tasks
- long summaries
- anything that belongs in Main Notes or Source Materials

At the end, ask Pedro to approve, reject, or edit the proposal.
```
