# 08 - Session Handoff

Use this at the end of a long AI session.

```text
Use `90. AI/skills/session-handoff.md`.

Create a session handoff for this work session.

Save it in:

`90. AI/history/`

Include:

- session goal
- what was done
- files created
- files updated
- files backed up
- decisions made
- pending Pedro approval
- open questions
- next steps
- important context for the next AI session
- related files

This is not memory.

If something should become memory, create a separate memory proposal.
```
